Internet of Things (IoT)

During my degree I self-defined two Internet/Intranet of Things projects, involving creating a new networking protocol from the ground-up, to enable battery-powered embedded devices to efficiently form an Intranet of Things (iot), securing this protocol using both asymmetric and symmetric cryptography schemes, as well as integrating the iot into a high-performance complex event processing engine called the Homework Cache.

I've implemented to the protocol on several platforms (secure/non-secure), all available on github.

  • TelosB Contiki OS
  • TelosB TinyOS (Secure)
  • Arduino Platform (Secure in-progress)

My current plans are to update the Arduino port, including various fixes and basic symmetric security; extend integration support, including porting the controller role to a desktop application, as well as providing a user interface for the event engine; plus, adding intelligence to the closed-loop control.

Other Projects

I've also got several other projects I've worked on/working on/planning:

Control Room - A HTML5 + Canvas project allowing users to easily plan and design model train tracks. Also uses a custom Javascript parser for reading in existing track component files. Future plans to integrate track design with actual control of live trains and layout features via Arduino + digital train control.

VLC Sync - A mini-project using libVLC and SDL to create a synchronised network video player in C.

Wireless 2.4GHz Camera Viewer - Uses libVLC + SDL to create a dedicated camera viewer for Linux, supporting channel switching (1-4). Future plans to integrate network control and viewing.

MultiTimer- A timer multiplexer for the Arduino, using interrupts to support multiple simultaneous event timers commonly found on other platforms (Contiki Event Timer). Created to support the Arduino iot port.

Remote Plugs library - An Arduino library for controlling Maplin Remote Control plugs, using a 433mhz transmitter. Created as a possible accessory for the iot protocol.
